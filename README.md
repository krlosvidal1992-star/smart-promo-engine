# smart-promo-engine 📈

Herramienta de escritorio desarrollada en Python para la optimización de ventas y análisis de rentabilidad financiera, diseñada específicamente para el rubro de sublimación y retail.

## ✨ Funcionalidades
- **Cálculo de Rentabilidad Real:** Deducción automática del IVA (10%) sobre el precio de venta.
- **Análisis de Estado:** Clasificación de productos (OK, Estrella o Pérdida) según el margen neto.
- **Asistente de Marketing:** Generación de sugerencias promocionales con concordancia gramatical automática.
- **Interfaz Gráfica:** entorno visual amigable desarrollado con Tkinter.

## 🛠️ Tecnologías
- **Lenguaje:** Python 3.11+
- **Librerías:** Tkinter (GUI), Messagebox (Alertas)
